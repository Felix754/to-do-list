A minimal **browser-based To-Do List** built with pure HTML, CSS, and JavaScript. The goal of the project is to try base frontend technologies.

## Features
- Add tasks/task-groups
- Remove tasks/task-groups
- Mark as completed
- Simple clean UI
- Saving all data to browser cache

## Installation
```bash
git clone https://github.com/Felix754/to-do-list.git
open index.html
```
